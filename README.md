# 🚕 OLA Ride Analytics Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing OLA ride booking data to generate meaningful business insights using **SQL, Power BI, and Data Analytics techniques**.

The dashboard provides insights into ride performance, customer behavior, vehicle performance, revenue analysis, cancellation patterns, and rating analysis.

The objective of this project is to help business stakeholders understand ride trends and make data-driven decisions.

---

# 📊 Dashboard Features

The Power BI dashboard contains multiple pages covering different analytical areas.

## 1. Overall Analysis

### Key Metrics:
- Total Rides
- Successful Bookings
- Total Booking Value
- Total Distance Travelled
- Average Distance Travelled
- Average Customer Rating
- Average Driver Rating

### Visualizations:
- Ride Volume Over Time
- Booking Status Breakdown

Insights:
- Understand overall ride performance
- Analyze booking success and cancellation trends
- Track ride growth patterns
